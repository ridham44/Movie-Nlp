Our Movie Recommendation System is an innovative AI-powered project developed as part of the Artificial Intelligence course. It leverages Machine Learning (ML) and Natural Language Processing (NLP) techniques to recommend movies based on user mood, keywords, or actor preferences. The system includes an Emotion Classifier that analyzes text input (e.g., ‚ÄúI want something adventurous with unexpected twists‚Äù) and maps it to emotions such as happy, sad, curious, or excited. These detected moods are then linked to suitable movie genres, ensuring a more personalized movie discovery experience.


To build this system, we used a curated movie dataset integrated with metadata from the Hugging Face datasets API, along with a custom emotion dataset (emotion_dataset_raw.csv) for training the mood classifier. The backend is powered by Flask with REST APIs, and the frontend is designed with modern HTML, CSS (Glassmorphism design), and JavaScript for interactivity. This combination of technologies demonstrates how AI and ML can be applied to real-world entertainment systems, making movie recommendations smarter, adaptive, and user-friendly.
